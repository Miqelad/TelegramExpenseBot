Определи параметры отчета по категории или теме расходов.

Возвращай только JSON.
Без markdown.
Без пояснений.

Формат:

{
  "category": "КАТЕГОРИЯ",
  "descriptionQuery": "поисковый текст",
  "months": число,
  "month": "месяц",
  "dateFrom": "yyyy-MM-dd",
  "dateTo": "yyyy-MM-dd"
}

Правила:
- category должна быть одной из категорий ниже или null
- descriptionQuery используй для конкретного слова/сервиса/места из запроса
- если пользователь просит широкую категорию, descriptionQuery может быть null
- если пользователь просит конкретную тему внутри категории, заполни и category, и descriptionQuery
- если пользователь пишет "кофе", category = "COFFEE", descriptionQuery = null
- если пользователь пишет "яндекс такси", category = "TAXI", descriptionQuery = "яндекс такси"
- если пользователь пишет "chatgpt", category = "DIGITAL_SERVICES", descriptionQuery = "chatgpt"
- если категорию определить нельзя, category = null, а descriptionQuery заполни словами из запроса
- если пользователь просит отчет за несколько месяцев, заполни months
- если пользователь указал месяц, заполни month на английском: JANUARY, FEBRUARY, MARCH, APRIL, MAY, JUNE, JULY, AUGUST, SEPTEMBER, OCTOBER, NOVEMBER, DECEMBER
- если даты не указаны, оставь months, month, dateFrom, dateTo равными null
- если пользователь не указал год, используй текущий год
- текущий год: 2026
- не добавляй лишние поля
- ответ должен быть валидным JSON объектом
