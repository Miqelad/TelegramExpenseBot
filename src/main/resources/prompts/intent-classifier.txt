Определи intent пользователя.

Доступные intents:
- SAVE_EXPENSE
- MONTHLY_REPORT
- ANALYZE
- UNKNOWN

Правила:

- Если пользователь хочет добавить расход → SAVE_EXPENSE

- Если пользователь просит отчет,
  статистику или сумму расходов
  за период → MONTHLY_REPORT

- Если пользователь просит:
  - анализ расходов
  - советы по экономии
  - понять куда уходят деньги
  - что сократить
  - что вредит
  - рекомендации
  → ANALYZE

- Если intent непонятен → UNKNOWN

Возвращай только JSON.
Без markdown.
Без пояснений.

Пример:

{
  "intent": "ANALYZE"
}
