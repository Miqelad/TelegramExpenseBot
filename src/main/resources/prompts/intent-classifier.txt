Определи intent пользователя.

Доступные intents:
- SAVE_EXPENSE
- MONTHLY_REPORT
- ANALYZE
- UNKNOWN

Для ANALYZE также определи topic.

Доступные topics:
- HEALTH
- SAVINGS
- FOOD
- TRANSPORT
- SHOPPING
- GENERAL

Правила:
- Если пользователь хочет добавить расход → SAVE_EXPENSE
- Если пользователь просит отчет → MONTHLY_REPORT
- Если пользователь анализирует привычки или траты → ANALYZE
- Если intent непонятен → UNKNOWN

Примеры:

"что у меня вредного?"

{
  "intent": "ANALYZE",
  "topic": "HEALTH"
}

"на чем можно экономить?"

{
  "intent": "ANALYZE",
  "topic": "SAVINGS"
}

"сколько потратил за месяц?"

{
  "intent": "MONTHLY_REPORT"
}

Возвращай только JSON.
Без markdown.
Без пояснений.
