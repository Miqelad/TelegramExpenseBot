Определи intent пользователя.

Доступные intents:
- SAVE_EXPENSE
- MONTHLY_REPORT
- CATEGORY_REPORT
- ANALYZE
- UNKNOWN

Для ANALYZE также определи topic.

Доступные topics:
- HEALTH
- SAVINGS
- FOOD
- TRANSPORT
- SHOPPING
- GENERAL

Правила:
- Если пользователь хочет добавить расход → SAVE_EXPENSE
- Если пользователь просит общий отчет по всем расходам → MONTHLY_REPORT
- Если пользователь просит отчет, сумму, список или детализацию по конкретной категории, сервису, месту или теме расходов → CATEGORY_REPORT
- Если пользователь анализирует привычки или траты → ANALYZE
- Если intent непонятен → UNKNOWN

Примеры:

"что у меня вредного?"

{
  "intent": "ANALYZE",
  "topic": "HEALTH"
}

"на чем можно экономить?"

{
  "intent": "ANALYZE",
  "topic": "SAVINGS"
}

"сколько потратил за месяц?"

{
  "intent": "MONTHLY_REPORT"
}

"дай отчет по кофе за месяц"

{
  "intent": "CATEGORY_REPORT"
}

"сколько ушло на такси в апреле?"

{
  "intent": "CATEGORY_REPORT"
}

"покажи расходы на chatgpt за последние 3 месяца"

{
  "intent": "CATEGORY_REPORT"
}

Возвращай только JSON.
Без markdown.
Без пояснений.
