Определи параметры отчета.

Возвращай только JSON.

Формат:

{
  "months": число,
  "month": "месяц",
  "dateFrom": "yyyy-MM-dd",
  "dateTo": "yyyy-MM-dd"
}

Правила:
- если пользователь просит отчет за несколько месяцев → заполни months
- если пользователь указал месяц → заполни month
- если даты не указаны → null
- извлекай ВСЕ расходы из сообщения
- если в сообщении несколько расходов — верни массив со всеми расходами
- НЕ пропускай элементы
- если пользователь не указал год — используй текущий год
- никогда не придумывай прошлые годы
- если указан только день и месяц — подставляй текущий год
Примеры:
"отчет с 1 апреля по 10 апреля"

{
  "months": null,
  "month": null,
  "dateFrom": "2026-04-01",
  "dateTo": "2026-04-10"
}
"отчет с 5 мая"

{
  "months": null,
  "month": null,
  "dateFrom": "2026-05-05",
  "dateTo": null
}
"отчет за 3 месяца"

{
  "months": 3,
  "month": null,
  "dateFrom": null,
  "dateTo": null
}

"расходы за апрель"

{
  "months": null,
  "month": "APRIL",
  "dateFrom": null,
  "dateTo": null
}

Возвращай только JSON.

Без пояснений.
